# Project: FinanceApp

## Vision
Empowering users (from students to wealth-builders) to achieve specific savings goals through AI-optimized budgeting and actionable investment insights.

## Core Value
High-fidelity financial monitoring with a privacy-first, offline-first approach (WatermelonDB + Local LLM).

## Technical Stack
- **Frontend**: React Native (Expo)
- **Database**: WatermelonDB (SQLite local)
- **Sync**: Supabase
- **AI**: Remote LLM (v1), Local LLM/llama.rn (v2)
- **Styling**: NativeWind (Tailwind) / Premium Glassmorphism
